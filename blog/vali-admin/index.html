Vali Admin

A free, modular and easy to customize dashboard theme built using Bootstrap, SASS and Pug JS.

DemoDownloadGitHub

Getting started

Vali is a responsive admin theme built with bootstrap 3, SASS and jade. It's fully modular so you don't have to copy the code which you don't need. Modularity is achieved using component based stylesheet and javascript.

If you like the theme do share it with your friends. If you have any issues or feedback regarding theme let me know in comments . Pull requests are always welcome.
